## Project goals

Ideas for the proposal.

* Engage with bio community, look at their workflows, see what they
  hack together using LLMs—and make those pathways more efficient.

* AI-ready documentation infrastructure: standardize MyST
  specification for machine sharing; ensure that documents are
  optimally organized for machine consumption, API discovery, and code
  generation.

* RunLLM / Jupyter-AI integration: provide natural language query
  interfaces, so that users may ask questions about docs: "how do I do
  X", "please provide an initial pipeline for Y", etc.

* Improved plugin and interactive widget support. Widgets that
  communicate, example widgets for the life sciences domain, improve
  plugin infrastructure + stage hooks.

* In collaboration with Jupyter Hub:

  * Rapid publishing workflows: develop lightweight rendering and
    publishing servers tailored for JupyterHub environments.

  * GPU-enabled remote kernel execution: build native support for
    remote kernel execution with GPU integration, allowing
    computation-heavy code blocks within MyST documents to run
    seamlessly on cloud (hub) infrastructure.
