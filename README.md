Renaissance Philanthrophy has a [call for proposals in Open Source ecosystems](https://os4science.org/funding_opportunity/os4ls/). This is similar to the EOSS grants that CZI used to run, and similarly this call has a particular interest in open source work that benefits AI workflows and open source.

We'd like to use this advance some bigger things that we want to accomplish in the Jupyter Book / MyST ecosystem. We're using this repository to coordinate themes, people, and the proposal itself.
